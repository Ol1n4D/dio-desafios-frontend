# 🏦 NeoFin - Validador de Produtos Financeiros

![Badge em Desenvolvimento](https://shields.io)
![GitHub License](https://shields.io[SEU_USUARIO]/[NOME_DO_REPOSITORIO]?style=for-the-badge)

Interface moderna de uma fintech criada para resolver o desafio de projeto **"Conhecendo os Pilares do Frontend"** da plataforma **DIO (Digital Innovation One)**. O sistema simula a validação e identificação rápida de produtos financeiros a partir de códigos específicos fornecidos por parceiros comerciais.

🌐 **[Clique aqui para acessar o projeto no ar](https://[SEU_USUARIO].github.io/[NOME_DO_REPOSITORIO]/)**

---

## 📌 Funcionalidades

- **Validação de Código:** Identifica instantaneamente o tipo de produto financeiro correspondente.
- **Tratamento de Erros:** Exibe um alerta visual dinâmico caso o código seja desconhecido.
- **Design Responsivo:** Interface otimizada e limpa (Estilo Dark Mode) adaptável para computadores e celulares.
- **Filtro de Entradas:** Limpeza automática de espaços em branco antes da consulta.

## 📊 Regra de Negócio (Tabela de Códigos)


| Código | Produto Financeiro |
| :--- | :--- |
| **CC** | Cartao de Credito |
| **CD** | Conta Digital |
| **EMP** | Emprestimo |
| **INV** | Investimento |
| *Outros* | Produto desconhecido |

## 🛠️ Tecnologias Utilizadas

Para a construção deste ecossistema frontend, apliquei os três pilares fundamentais da web:

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização moderna, variáveis CSS para gerenciamento de cores e Flexbox para alinhamento.
- **JavaScript (ES6):** Manipulação de eventos do DOM e estrutura condicional `switch` para validação lógica de dados.
